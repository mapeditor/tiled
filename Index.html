Entity.create_entity() Create a new entity and return a reference to it.

Entity.add_sprite_component(entity, alias, width, height, zOrder) Add a SpriteComponent to the entity using a texture alias and basic sprite layout.
Entity.add_audio_component(entity[, clip, playOnAwake]) Add an AudioPlayer component to the entity. Optional: clip (string) and playOnAwake (bool, default false).

Entity.has_fysics_component(entity) Returns true if the entity has a FysicsBody component.

UI

The UI table offers both immediate-mode drawing and managed UI elements via the UIManager.

FunctionDescription
UI.Add_label(x, y, w, h, text) Add a managed label at (x, y) with size (w, h) and the given text.
UI.draw_label(text, w, h, x, y, font) Draw a label immediately at (x, y) with size (w, h). font is currently unused/placeholder.
UI.draw_button(text, w, h, x, y) Draw a button immediately; returns true if clicked this frame.
UI.add_button(x, y, w, h, text, onClickFn) Add a managed button at (x, y) with size (w, h). When clicked, calls the provided Lua callback onClickFn().
UI.add_slider(label, value, w, h, x, y, min, max) Draw a slider and return the updated value. Label is shown next to the slider; value is clamped between min and max.

Audio

Audio is controlled through an AudioPlayer component attached to an entity. In Lua this is exposed via the AudioComponent table. A typical flow: 


local e = Entity.create_entity()
Entity.add_audio_component(e, "some_clip_alias", false)
local audio = Entity.get_audio_component(e)

-- Then call AudioComponent functions with this entity:
AudioComponent.play(e)

AudioComponent

FunctionDescription
AudioComponent.play(entity[, fadeMs[, loops]]) Play the attached audio. If only entity is provided, plays normally. If fadeMs is given, fades in over that many milliseconds. If loops is also provided, plays with the specified loop count.
AudioComponent.pause(entity) Pause playback.
AudioComponent.resume(entity) Resume playback if paused.
AudioComponent.terminate(entity) Stop playback completely.
AudioComponent.is_playing(entity) Returns true if audio is currently playing.
AudioComponent.play_one_shot(entity, clipName, volumeScale) Play a one-shot sound by name with a volume scale factor (e.g. 1.0 = normal, 0.5 = half volume).
AudioComponent.change_volume(entity, value) Set the volume. Typical internal range is 0–128, where 0 is silent and 128 is max.
AudioComponent.toggle_mute(entity) Toggle mute on/off for this audio component.
AudioComponent.get_volume(entity) Get the current volume value.
AudioComponent.get_mute(entity) Returns true if audio is currently muted.
AudioComponent.get_clip(entity) Get the current sound clip alias assigned to this component.
AudioComponent.get_play_on_awake(entity) Returns true if the clip will auto-play when the component is created.
