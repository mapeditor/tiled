--- layout: post title: Tiled Qt 0.3.0 released date: '2009-11-13T23:24:00.000+01:00' author: Thorbj√∏rn Lindeijer tags: - release modified_time: '2011-09-27T23:35:37.894+02:00' blogger_id: tag:blogger.com,1999:blog-1263769677734306791.post-8736792845175006601 blogger_orig_url: http://blog.mapeditor.org/2009/11/tiled-qt-030-released.html --- The Qt version of Tiled is still catching up, but it's catching up fast! The main new feature in this release is support for cut, copy and paste. This is noteworthy, since it will even work from one map to another or between Tiled instances. The full list of changes:
  • Added a tile selection tool
  • Added support for cut, copy and paste
  • Added current cursor position to the status bar
  • Added keyboard shortcuts to switch tools
  • Added scrolling the map view with middle mouse button
  • Snap objects to the grid when Ctrl is pressed
